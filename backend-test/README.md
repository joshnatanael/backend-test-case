# Backend Test - Joshua Natanael

## Running the app

### Setup database
- edit .env.production file with database data

### Run application
```bash
# seeding
$ npm run db:seed:all

# build
$ npm run build

# run
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
